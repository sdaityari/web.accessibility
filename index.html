WAI ARIA

Web Accessibility Initiative - Accessible Rich Internet Applications

How does it work?

  • ARIA defines ways to make dynamic web applications accessible
  • Assigns roles and states to web elements using the role and state attributes
  • Supported by all modern browsers and screen readers
  • Not required for simple applications if you use proper markup

Best resources are available on the Mozilla Developer Network.

Final words

  • Making your website accessible is boring
  • Work as you are building your website
  • Work with designers to make sure the designs are accessible
  • Accessibility is time dependent
  • If your site was accessible last year, it might not be adequately accessible now!
